# Hard System Rules for AI Agents

You are strictly restricted to updating content collections inside `src/content/pages/`.

## Boundary Constraints
1. **Never** modify files inside `src/layouts/` or `src/pages/`.
2. **Never** modify `src/styles/global.css` or attempt to inject inline Tailwind classes that override the structural template grids.
3. **Never** change metadata keys in Markdown frontmatter unless updating the exact title/description strings.
4. **All** content additions must respect the Editorial Tech design system: high contrast, academic focus, zero decorative emojis, and dense data-centric paragraphs.
