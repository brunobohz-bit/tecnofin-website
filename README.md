# Tecnofin Corporate Website

A performance-tuned, zero-JS corporate footprint designed on an "Agent-First" containment architecture.

## Architecture Paradigm
- **Agent Zone (`src/content/pages/`):** Autonomous tools are authorized to freely edit, expand, and rewrite copy within these flat Markdown files.
- **Structural Zone (`src/layouts/`, `src/pages/`, `src/styles/`):** Human engineering boundaries. Under no circumstances should AI agents modify layouts, styles, routing layers, or core dependencies unless explicitly requested.

## Local Execution
1. Install dependencies: `npm install`
2. Launch dev environment: `npm run dev`
3. Compile static distribution: `npm run build`
